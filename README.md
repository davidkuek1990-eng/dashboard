# Mini Authentication Dashboard

A simple frontend authentication demo project built to simulate login flow, protected routing, and dashboard access. This project demonstrates basic application structure, authentication flow, and frontend state management.

---

## Features

- User login interface (simulated authentication)
- Protected dashboard route after login
- Session persistence using localStorage
- Logout functionality with session clear
- Simple dashboard layout after authentication
- Route guarding for unauthenticated users

---

## Tech Stack

- Next.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router (if applicable)

---

## Project Structure

- **Login Page** – Entry point for user authentication
- **Auth Logic** – Simulated authentication flow
- **Protected Routes** – Restricts access to authenticated users
- **Dashboard** – Main protected interface after login

---

## How It Works

1. User enters login credentials (mock validation)
2. System simulates authentication and generates a fake session/token
3. Session is stored in localStorage
4. User is redirected to the dashboard
5. Session persists on page refresh
6. Logout clears session and redirects to login page

---

## Purpose of Project

This project was built to demonstrate:

- Frontend authentication flow understanding
- Route protection and navigation control
- State management using localStorage
- Basic frontend application architecture
